
#---------------------------------------------------------------------------
#  IconFont - Masamune Framework
#---------------------------------------------------------------------------

The Masamune Framework is an asset series that conveniently extends Unity.
This asset primarily adds the following features to Unity and UniryEditor:

1. Using Icon Fonts
You can use the icon fonts that you often see on modern websites and flat-designed apps.
Imported icon fonts will be available in uGUI, editor extensions, etc.
Easy-to-use components are included so they are available as soon as the package is installed.

2. UIElements tags for icons
Tags such as UIElements buttons (IconButton) and labels (Icon) using icon fonts will be available.

3. Icon Search Window
You can search the list for imported icon fonts.
Launch it from the menu, Masamune- > IconFont.

4. FontAwesome Icon Fonts Included
The famous FontAwesome icon font is included by default.
You can also use various icon fonts by creating your own importer.
https://fontawesome.com/

5. Common Masamune Framework Features
In addition to the above, other useful features are provided.
Please see the following page for details.
https://mathru.net/unity/2905/

インポートされたアイコンフォントを一覧から検索できます。

有名なFontAwesomeのアイコンフォントをデフォルトで同梱しています。

Masamune Framework 는 Unity 을 편리하게 확장 할 자산 시리즈입니다.

1. 아이콘 글꼴 사용
현대적인 Web 사이트 및 평면 디자인의 응용 프로그램 등에서 흔히 볼 아이콘 폰트를 사용할 수 있습니다.
가져온 아이콘 글꼴 uGUI 및 편집기 확장 등으로 사용할 수 있습니다.
사용하기 쉬운 구성 요소를 포함하고 있기 때문에 패키지 도입 후 즉시 사용할 수 있습니다.

2. 아이콘 용 UIElements 태그
아이콘 폰트를 이용한 UIElements 버튼 (IconButton)과 라벨 (Icon) 등의 태그를 이용할 수있게됩니다.

3. 아이콘 검색 창
가져온 아이콘 글꼴 목록에서 찾을 수 있습니다.
메뉴 Masamune-> IconFont에서 시작하십시오.

4. FontAwesome 아이콘 글꼴을 포함
유명한 FontAwesome 아이콘 글꼴을 기본적으로 포함하고 있습니다.
가져 오기를 자작하면 다양한 아이콘 폰트를 사용할 수 있습니다.
